# [Fastbot Setup Project](https://github.com/kuralme/fastbot)

Fastbot is a mobile robot platform developed as part of the *Robotics Masterclass*. It combines hardware assembly with advanced software integration using ROS 2, enables practical experience in navigation, sensor processing, and robot control.

![Fastbot]({{ site.url }}{{ site.baseurl }}/assets/fastbot.gif)

## Key Features

- **Raspberry Pi 4 Core** — Supports Python, C++, and ROS 2 programming environments.

- **Differential Drive System** — Provides odometry through wheel encoders for precise movement tracking.

- **360º LiDAR Sensor** — Captures distance and intensity data up to 12 m for mapping and obstacle avoidance.

- **Raspberry Pi Camera** — Enables computer vision tasks such as object detection and SLAM visualization.

- **Dockerized ROS 2 Environment** — Simplifies deployment, dependency management, and reproducibility.

- **Hot-Swappable Battery System** — Ensures uninterrupted operation during extended testing sessions.
